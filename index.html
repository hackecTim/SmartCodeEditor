```html SmartCode — Choose a Mode

SmartCode Editor

Choose how you want to use the code editor.

PROJECT Mode

A full editor with a toolbar, file tabs, status bar, diagnostics and support for multiple files. This mode is suitable for using the editor as a standalone project environment.

Open PROJECT Mode

FOLDER Mode

Opens a folder picker on startup. The user selects a local folder, and the editor loads files from that folder and its subfolders. This mode is suitable for working with an existing project.

Open FOLDER Mode

SINGLE Mode

An embedded editor without file tabs and toolbar. This mode is intended for integrating the editor into another web application, for example ALGator.

Open SINGLE Mode
Before using the editor, make sure that the Docker LSP server is running at http://localhost:3000.
```
